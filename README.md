# Wallet Goals PWA

Локальный трекер целей и баланса.

## Как запустить

1. Загрузить все файлы в GitHub репозиторий.
2. Settings → Pages.
3. Source: Deploy from a branch.
4. Branch: main.
5. Folder: /(root).
6. Save.

После появления ссылки открыть ее на iPhone в Safari:
Поделиться → На экран «Домой».

Данные хранятся локально в браузере устройства.
